## Server commands:
```
account adduser <username> <password>  
account addadmin <username> <password>  
account remove <username>  
account changepw <username> <old password> <new password>  
```

## Client commands:
```
control shutdown <minutes>  
control restart <minutes>  
control abort  

volume up
volume down
volume mute

media play
media previous
media next
```
